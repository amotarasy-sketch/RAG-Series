# Demo books

This folder contains sample FB2 books used by default when the project is run locally.

The included works are classic Russian literature in the public domain:

- Александр Пушкин — «Евгений Онегин»
- Иван Тургенев — «Отцы и дети»
- Лев Толстой — «Анна Каренина»

You can replace these files with your own `.fb2` books, or override the folder with `BOOKS_DIR`.
